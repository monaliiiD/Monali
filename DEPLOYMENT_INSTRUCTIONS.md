# How to Host Your Website

Success! Your code has been pushed to GitHub: [https://github.com/monaliiiD/Monali](https://github.com/monaliiiD/Monali)

## Final Step: Enable GitHub Pages

1.  Go to your **GitHub Repository Settings**:
    [https://github.com/monaliiiD/Monali/settings/pages](https://github.com/monaliiiD/Monali/settings/pages)

2.  Under **Build and deployment** > **Source**, select **Deploy from a branch**.

3.  Under **Branch**, select `main` and ensure the folder is `/ (root)`.

4.  Click **Save**.

## Your Website URL
Wait about 1-2 minutes, and your website will be live at:
**[https://monaliiiD.github.io/Monali/](https://monaliiiD.github.io/Monali/)**
